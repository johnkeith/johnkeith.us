For me, development is about building solutions that delight. I enjoy the whole process of bringing an app to life – from first rough sketchs of a concept to final touches on the code that make for an engaging user experience. Below is a sampling of my work.

GitCard: App for displaying your best work on GitHub. Here are two example profiles from site users: example one, example two. (Rails, Haml, Sass, AJAX)

Scrumptiously Skinny: Food blog hosted on Wordpress with a minimal, elegent design. (Wordpress, PHP, HTML, CSS, Javascript)

TEDx Hammond School: TEDx event site for the Hammond School. (Bootstrap, HTML, CSS, Javascript)

UnoDos: Javascript game inspired by 2048. (Javascript, jQuery, HTML, CSS)

So Fresh and So Green: A straightforward Octopress theme: it focuses on excellent readability. (Octopress, HTML, Sass)

Bold and Blue: Bold and Blue is an Octopress theme designed to be bold, but minimal. (Octopress, HTML, Sass)
